# fluxible.io

The doc site for [Fluxible](https://github.com/yahoo/fluxible).


## Setup

```bash
$ npm install
```


## Run the app

```bash
$ npm run build
$ npm start
```

Open http://localhost:3000


## Run in dev mode

```bash
$ npm run dev
```

This will use `nodemon` and `webpack` to watch for changes and restart and
rebuild as needed.

Open http://localhost:3000
